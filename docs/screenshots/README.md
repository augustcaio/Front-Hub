# Capturas de Tela

Esta pasta contém as capturas de tela da aplicação Front-Hub.

## Imagens Necessárias

Para completar a documentação, adicione as seguintes capturas de tela:

1. **login.png** - Tela de login
2. **dashboard.png** - Dashboard principal
3. **devices-list.png** - Lista de dispositivos
4. **device-detail.png** - Detalhes do dispositivo
5. **alerts.png** - Painel de alertas

## Formato Recomendado

- **Formato**: PNG ou JPG
- **Resolução**: Mínimo 1280x720 (preferencialmente 1920x1080)
- **Tamanho**: Máximo 2MB por imagem (comprima se necessário)

## Como Adicionar

1. Execute a aplicação
2. Navegue até a tela desejada
3. Tire uma captura de tela
4. Salve o arquivo nesta pasta com o nome correspondente
5. Atualize o README.md principal se necessário

## Ferramentas Úteis

- **Windows**: Snipping Tool, Windows + Shift + S
- **Linux**: Gnome Screenshot, KDE Spectacle
- **Mac**: Cmd + Shift + 4
